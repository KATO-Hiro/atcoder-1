## Environments

- Ubuntu 18.04.2 LTS
- Python: 3.4.3, 3.8.2
- pyenv-virtualenv
- VS Code

```bash
# Python
pyenv install 3.4.3
pyenv install 3.8.2

# virtualenv
pyenv virtualenv 3.4.3 atcoder-3.4.3
pyenv virtualenv 3.8.2 atcoder-3.8.2

# python packages
cd env-3.4.3
python -m pip install -r requirements.txt
cd env-3.8.2
python -m pip install -r requirements.txt
```
